Pour compiler le kernel sous gentoo :

cd /usr/src/linux

make menuconfig

make && make modules_install

make install

Le kernel est maintenant compilé.

Maintenant il faut mettre à jour grub. :

grub2-mkconfig -o /boot/grub/grub.cfg
