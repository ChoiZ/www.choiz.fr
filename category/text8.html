Voir les fichiers cachés sur Lion

Posté le mer. 02 novembre 2011 • Tags : file, osx

Dans un terminal :

defaults write com.apple.Finder AppleShowAllFiles TRUE

killall Finder

Ouvrir une fenêtre du Finder et vous voyez les fichiers cachés.
