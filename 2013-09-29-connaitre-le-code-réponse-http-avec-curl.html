Connaitre le code réponse HTTP avec curl

Posté le dim. 29 septembre 2013

Pour connaitre le code réponse HTTP il suffit de faire :

curl -IL url

Voici l'exemble avec tumblr.com :

curl -IL http://tumblr.com
